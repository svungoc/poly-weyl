Module type Math.VSpace

Vector spaces over a field

A module of type VSpace can be used as an AbelianGroup. VSpace is simply an alias to Module; it can be used when the scalar ring is a field.

type scalar

The scalar type should in principle include Rng.t, but this signature does not enforce it here.
